# App Bancaria - Práctica Java

Proyecto inicial para practicar lógica de objetos, encapsulamiento y colecciones.

### Funcionalidades principales:
* **Gestión de Cuentas:** Registro de nombre, dirección y tipo de cuenta (Ahorro, Corriente, Sueldo).
* **ID Autoincremental:** Generación automática de identificadores únicos.
* **Sistema de Transacciones:** Depósitos y transferencias entre cuentas con validación de saldo.
* **Historial:** Registro detallado de movimientos por usuario.

### Reglas técnicas:
- Uso exclusivo de `ArrayList`.
- Lógica de negocio centralizada en un `GestorBancario`.
- Implementación de Enums para tipos de cuenta.

### Reportes:
- Balance general de todos los usuarios activos.
- Listado detallado de transacciones por cliente.
